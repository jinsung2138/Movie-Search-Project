# Movie-Search-Site

주요 작업
1. fetch를 통한 TMDB API 데이터 GET
  - 영화 ID(id)
  - 제목(title)
  - 평점(vote_average
  - 개요(overview)
  - 포스터 이미지 주소(poster_path)

2. TMDB API 데이터 JSON 가공 및 출력
  - 영화 ID(id)
  - 제목(title)
  - 평점(vote_average
  - 개요(overview)
  - 포스터 이미지 주소(poster_path)

3. 카드 클릭 시 영화 ID값 alert창 띄움

4. 영화 검색 기능
  - 키보드 Enter 또는 검색 버튼 클릭 시 영화 검색

5. 페이지 랜딩 또는 새로고침 시 Input에 자동 커서 이동
